# Cryptography Implementations in Python

A comprehensive collection of cryptographic algorithms and protocols implemented in Python, ranging from classical ciphers to modern cryptographic primitives.

## Table of Contents
- [Overview](#overview)
- [Algorithms Included](#algorithms-included)

## Overview

This repository contains Python implementations of various cryptographic algorithms completed as part of the lab work for the Information Security subject studied in the 5th semester of BIM. These implementations demonstrate fundamental concepts in cryptography and information security.


## Algorithms Included

### Classical Ciphers
1. **Caesar Cipher** (`1_caesar.py`) - Simple substitution cipher
2. **Playfair Cipher** (`2_playfair.py`) - Digraph substitution cipher
3. **Affine Cipher** (`3_affine.py`) - Monoalphabetic substitution cipher
4. **Hill Cipher** (`4_hill.py`) - Polygraphic substitution cipher using linear algebra
5. **Rail Fence Cipher** (`5_railfence.py`) - Transposition cipher
6. **Vigenère Cipher** (`6_vigenere.py`) - Polyalphabetic substitution cipher

### Number Theory & Primality Testing
7. **GCD Calculator** (`7_gcd.py`) - Euclidean algorithm implementation
8. **Modular Arithmetic** (`8_modularArithmetic.py`) - Modular operations
9. **Fermat Primality Test** (`9_Fermat.py`) - Probabilistic primality test
10. **Miller-Rabin Test** (`10_miller.py`) - More robust primality test

### Modern Cryptography
11. **DES** (`11_des.py`, **'11_DES1.py'**) - Data Encryption Standard *`11_des.py` is just an example implementation and '11_DES1.py' is the complete implementation of DES algorithm
12. **Diffie-Hellman** (`12_diffieHellmen.py`) - Key exchange protocol
13. **RSA** (`13_rsa.py`) - Public-key cryptosystem
14. **MD5** (`14_md5nolib.py`) - Message Digest Algorithm
15. **MD4** (`md4.py`) - Earlier hash function
16. **AES** (`aes.py`) - Advanced Encryption Standard
